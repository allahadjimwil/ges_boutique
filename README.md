# ges_boutique
## Description
Application web développée avec Laravel pour la gestion d'une boutique.
## Fonctionnalités
- Gestion des produits;
- Gestion des clients;
- Gestion des commandes;
- Gestion des factures;
- Interface web avec Laravel
## Technologies
- PHP
- Laravel
- MySQL
- HTML/CSS
- Blade
## Installation
1. Cloner le dépot
2. Installation des dépendances : '''bash, composer install '''
3. Copier '.env.example' vers '.env'.
4. Générer la clé: '''bash, php artisan key:generate'''
5. Configurer la base de données dans '.env'.
6. Exécuter les migrations: '''bash, php artisan migrate'''
7. Lancer le server: '''bash, php artisan serve'''
## Structure
-'app/Http/Controllers': Controleurs
-'app/Models': Modèles
-'database/migrations': Migratons
-'ressources/views': Vues
